# YouTube Tutorials 🧚‍♀️✨

Code repository for tutorials on my YouTube channel, where I teach software engineering to aspiring developers, career switchers, and builders who want real technical depth.

Each folder is a self-contained tutorial with its own README, code, and setup instructions.

## Tutorials

- [`coffeeshop-tutorial`](./coffeeshop-tutorial) — Build a coffeeshop landing page with HTML, CSS, and JavaScript.

_More tutorials coming soon._

## How to use these tutorials

1. Clone the repo:

```bash
   git clone https://github.com/platypusonmars/youtube-tutorials.git
```

2. `cd` into the tutorial folder you want to work on.
3. Follow the README inside that folder.
